## Overview
React.js를 이용하여 개발한 XChat 입니다. 🙂 <br/>


- React.js로 만든 프로젝트, 실시간 메신저 웹 애플리케이션입니다.
- 현재 개발중이고, 프론트엔드의 전반적인 부분을 맡았습니다.
- 디자인까지 직접 함으로써, 사용자 입장에서 고민하고, 원하는게 무엇인가를 가장 많이 생각하게 된 개발 과정이 아닌가 싶습니다.
- React.js에 대해 좀 더 깊이 있게 이해하는 과정이 되고있고, 가독성과 유지보수성이 높은 코드가 무엇인지 끊임없이 고민하게 되었습니다.
